# Banner 轮播图片说明

## 图片要求

顶部导航栏的轮播背景图片位于 `frontend-web/src/web-front/` 目录：

- `bachalpsee-4334666_1920.jpg` (banner1)
- `mountain-climbing-2124113_1920.jpg` (banner2)
- `mountains-4695049_1920.jpg` (banner3)

**注意**：当前使用的是 JPG 格式的图片，代码会自动从 `web-front` 目录加载。

## 图片规格

### 尺寸要求
- **宽度**：建议 1920px 或以上（支持响应式）
- **高度**：80px（桌面端）/ 50px（移动端）
- **格式**：PNG（支持透明）或 JPG
- **文件大小**：建议每个文件 < 500KB，确保加载速度

### 设计建议
1. **宽幅横向构图**：适合作为横幅背景
2. **左右延伸**：图片会被裁剪，左右各延伸10px营造溢出感
3. **上方区域**：建议上方区域颜色较浅，便于文字显示
4. **风格统一**：三张图片风格保持一致，轮播时更流畅

## 使用说明

1. 当前图片已放置在 `frontend-web/src/web-front/` 目录
2. 图片会自动加载，无需额外配置
3. 如需更换图片，请保持相同的文件名和路径

## 轮播效果

- **切换间隔**：2秒
- **切换动画**：淡入淡出（0.5秒）
- **交互控制**：
  - 鼠标悬停时暂停轮播
  - 鼠标离开后恢复轮播
  - 底部圆点指示器可点击切换

## 临时方案

如果暂时没有 banner 图片，系统会自动使用 CSS 渐变背景作为降级方案。

## 示例图片

可以参考以下风格的图片：
- 自然风景（山川、湖泊、云海）
- 城市风光（夜景、建筑）
- 抽象艺术（渐变、几何图形）
- 品牌主题（与平台风格一致）
