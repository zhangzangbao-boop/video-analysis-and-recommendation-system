# AI视频审核功能说明

## 功能概述

这是一个**简单易用、零成本**的AI视频审核方案，基于文本关键词过滤实现。

## 功能特点

✅ **零成本** - 无需第三方API，纯本地实现  
✅ **自动审核** - 视频上传时自动检测敏感内容  
✅ **智能建议** - 为管理员提供审核建议  
✅ **一键操作** - 支持一键采纳AI建议  
✅ **易于扩展** - 可轻松添加更多敏感词

## 工作原理

1. **视频上传时**：自动检测标题、描述、标签中的敏感词
2. **生成审核建议**：根据检测结果生成通过/驳回/人工审核建议
3. **保存到数据库**：审核建议保存在 `audit_msg` 字段
4. **管理员查看**：在审核界面显示AI建议，可一键采纳

## 审核规则

### 风险等级

- **低风险**：未检测到敏感词 → 建议通过
- **中风险**：检测到一般敏感词 → 建议人工审核
- **高风险**：检测到严重敏感词 → 建议驳回

### 当前敏感词库

**高风险词（直接驳回）**：
- 色情、暴力、恐怖、赌博、毒品

**一般敏感词（人工审核）**：
- 政治、政府、领导人、性、成人、裸露、血腥、杀人、诈骗

## 使用方法

### 1. 自动审核

视频上传后，系统会自动检测并生成审核建议，无需额外操作。

### 2. 查看AI建议

在管理员后台的"视频管理"页面：
- 打开待审核视频
- 查看"🤖 AI审核建议"提示框
- 根据建议颜色判断风险等级：
  - 🟢 绿色：建议通过
  - 🟡 黄色：建议人工审核
  - 🔴 红色：建议驳回

### 3. 一键采纳建议

- **通过**：点击"一键采纳AI建议（通过）"按钮
- **驳回**：点击"一键采纳AI建议（驳回）"按钮
- **忽略**：可以忽略AI建议，手动审核

## 扩展敏感词库

### 方法一：直接修改代码

编辑 `SimpleContentAuditUtil.java` 文件：

```java
// 添加高风险词
private static final Set<String> HIGH_RISK_WORDS = new HashSet<>(Arrays.asList(
    "色情", "暴力", "恐怖", "赌博", "毒品",
    "你的新敏感词"  // 在这里添加
));

// 添加一般敏感词
private static final Set<String> SENSITIVE_WORDS = new HashSet<>(Arrays.asList(
    // ... 现有词
    "你的新敏感词"  // 在这里添加
));
```

### 方法二：从文件读取（推荐扩展）

可以修改代码，从配置文件或数据库读取敏感词，实现动态管理。

## 注意事项

1. 当前实现为**文本关键词过滤**，无法检测：
   - 视频画面内容
   - 音频内容
   - 图片内容（封面除外，需扩展）

2. 敏感词库需要根据实际情况**定期更新**

3. AI建议仅供参考，**最终审核权在管理员**

4. 建议保留人工审核流程，AI作为辅助工具

## 测试方法

1. 上传包含敏感词的视频（如标题包含"色情"）
2. 查看审核建议是否正确显示
3. 测试一键采纳功能
